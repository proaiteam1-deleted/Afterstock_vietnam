# Design Principles

## Tone

AfterStock should feel premium, minimal, fast, and credible. It should read like a financial intelligence dashboard, not a game.

## Visual Direction

- Dark-mode first
- Subtle borders and layered panels
- Large numbers for market and sentiment signals
- Calm cyan, emerald, amber, and rose accents
- Mobile-first responsive layouts

## Avoid

- Gambling language
- Guaranteed prediction phrases
- Meme-stock chaos
- Childish retail-investor caricatures
- Too many saturated colors
- Playful mascot-led UI

## Copy Rules

- Say "sentiment", "crowding", "reference indicator", and "contrarian hint".
- Avoid "win", "bet", "jackpot", or "guaranteed".
- Repeat the financial-advice disclaimer in key surfaces.
