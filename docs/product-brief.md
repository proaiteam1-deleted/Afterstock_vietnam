# AfterStock Product Brief

AfterStock is an after-market retail-investor sentiment platform for Korean mega-cap stocks. The first version focuses only on Samsung Electronics and SK Hynix so the core experience stays immediate and credible.

## Positioning

AfterStock is not a stock-prediction game and does not provide buy or sell recommendations. It collects post-market up/down sentiment, visualizes crowding, and frames the result as a reference indicator for the next session.

## Core User Questions

- What stock am I looking at?
- Are retail users bullish or bearish?
- Is sentiment overheated?
- What is the AI-style interpretation?
- Can I vote up or down?

## First Release Scope

- Landing page
- Dashboard for two Korean mega-cap stocks
- Stock detail pages
- Community preview
- Local mock voting state
- Supabase-ready file boundaries

## Required Disclaimer

This is sentiment data for reference only, not financial advice.
